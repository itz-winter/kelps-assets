CSS Assets

Stylesheets used across Kelpwing's sites, available to reference or link directly.

Usage

Link a stylesheet directly from this CDN in your <head>:

<link rel="stylesheet" href="https://assets.kelpw.ing/style/ammonite.css">

The stylesheets are served with Access-Control-Allow-Origin: * and can be fetched cross-origin.
