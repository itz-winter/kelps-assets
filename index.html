Usage

All assets are served from https://assets.kelpw.ing/ and are publicly accessible. To link an asset directly, use its full URL:

https://assets.kelpw.ing/artwork/ammonite.png

CSS files support cross-origin access and can be linked directly in any page's <head>:

<link rel="stylesheet" href="https://assets.kelpw.ing/style/ammonite.css">
